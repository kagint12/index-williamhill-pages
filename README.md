# index-williamhill-pages

## 项目简介

`index-williamhill-pages` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面彼此独立，各不相关，仅作为静态 HTML 文件的集合存储于此。  
本仓库不针对任何特定域名、网站或服务，仅用于存档用途。

## 目录说明

```
index-williamhill-pages/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（如 CSS、JS、图片等）
├── README.md       # 本说明文件
└── LICENSE         # 仓库许可证
```

- `pages/` 目录下每个文件夹或文件对应一个独立的 HTML 页面。
- `assets/` 目录按需存放公共或页面特有的静态资源。
- 新增页面时，请在 `pages/` 下创建对应子目录或文件，并确保资源引用路径正确。

## 页面归档说明

- 所有页面均为静态 HTML，可直接通过浏览器打开。
- 页面之间不存在相互依赖，每个页面均可独立运行。
- 归档内容以历史资料或示例展示为目的，不保证实时更新。
- 页面中可能包含外部资源链接，这些链接的可用性由原始来源负责。
